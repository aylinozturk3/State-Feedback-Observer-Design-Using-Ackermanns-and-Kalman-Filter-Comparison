##Overview

This project investigates state-feedback controller design and state estimation techniques using:
-Ackermann’s Formula for pole placement
-Luenberger Observer
-Kalman Filter

The goal is to compare deterministic observer design with optimal stochastic estimation in terms of stability, convergence speed, and robustness to noise.

##Implementation

The entire project was implemented in MATLAB.
Main components include:
-State-space modeling of an LTI system
-State-feedback gain computation via Ackermann’s formula
-Observer gain design
-Kalman filter implementation
-Closed-loop simulations and performance comparison
